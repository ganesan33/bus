About Bus Management System

The **Bus Management System** is an interactive platform that allows users to search for buses, check their routes, and track their live locations on a map.

Features include:

  • Live bus tracking using GPS coordinates.
  • Search functionality for buses by number.
  • Display of bus routes with source and destination.
  • Nearby bus suggestions when seats are unavailable.
  • Ability to reset map and remove markers.

This system is developed using **Flask, Firebase, Leaflet.js, and JavaScript** for an interactive and efficient user experience.
